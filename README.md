# FocusFlow: Student Productivity Dashboard

FocusFlow is a responsive student productivity web application designed to help students manage tasks, organize study sessions, track habits, monitor screen time, and improve focus through an interactive dashboard.

## Features

### Dashboard Overview

- View task streaks and habit streaks
- Track completed tasks and focus sessions
- Monitor productivity score and screen time
- Receive personalized motivation prompts

### Task Management

- Add, complete, and delete tasks
- Track pending and completed work
- Visualize weekly task progress using charts

### Pomodoro Timer

- 25-minute focus timer with start, pause, and reset controls
- Track completed focus sessions
- View weekly focus statistics

### Habit Tracker

- Create and manage daily habits
- Mark habits as completed
- Track progress and streaks

### Screen Time Analyzer

- Log application usage manually
- Categorize usage as productive, entertainment, or social media
- Calculate productivity scores
- Analyze screen-time trends using charts

### Study Calendar

- Add study plans and goals by date
- Organize academic schedules
- Remove completed plans

### Notes and Profile Management

- Save study notes using browser storage
- Customize profile information
- Switch between dark and light themes

### Responsive Design

- Mobile-friendly dashboard layout
- Smooth animations and interactive UI components
